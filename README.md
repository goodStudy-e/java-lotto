## 필요한 기능
```
1. 로또 구입 금액을 입력받는기능 <br>
2. 로또 구입 금액에 맞게 로또를 발급하는 기능 <br>
3. 로또의 당첨 여부(등수, 당첨금)를 확인하는 기능 <br>
4. 총 수익률을 계산하는 기능 <br>
5. 로또 등수에 따른 당첨 갯수와 수익률 메시지 출력하는 기능 <br>
```
